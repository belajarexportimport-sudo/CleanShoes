1. PENDAHULUAN

Selamat datang di ShoeTrack! Aplikasi ini dirancang sebagai solusi All-in-One untuk bisnis laundry sepatu dan perawatan barang. Aplikasi ini membantu Anda mencatat order, mengelola stok, mengatur harga, dan memantau kinerja toko secara real-time.

BAB 1: SISTEM LOGIN & KASIR üîê

Agar setiap transaksi tercatat dengan jelas:

  • Masuk Kerja (Login): Masukkan Nama Panggilan Anda.
  • Tanda Tiap Transaksi: Setiap struk akan memuat nama Kasir/Admin yang bertugas.
  • Ganti Shift: Klik nama Anda di Dashboard -> Logout saat selesai bertugas.

BAB 2: DASHBOARD UTAMA üìä

Pusat kendali bisnis Anda:

  • Order Aktif: Jumlah sepatu yang masih ada di toko (belum diambil).
  • Omzet Hari Ini: Total pendapatan kotor hari ini.
  • Profit Bersih (Hijau): (Omzet - Pengeluaran). Indikator keuntungan riil harian.
  • Stok Packaging: Indikator merah jika stok (Dust Bag/Goodie Bag) menipis (< 5 unit).

BAB 3: MENCATAT ORDER BARU üëü

  • Data Pelanggan: Ketik nama pelanggan. Jika pelanggan lama, data akan muncul otomatis (Autocomplete).
  • Pilih Layanan: Klik layanan yang tersedia (Daftar layanan bisa diatur di Menu Pengaturan).
  • Estimasi Selesai: Default otomatis +3 hari. Bisa diubah manual dengan klik tombol "Edit".
  • Stok Otomatis: Sistem otomatis mengurangi stok Dust Bag/Goodie Bag saat order disimpan.

BAB 4: DAFTAR ORDER & STATUS üìù

Pantau status setiap sepatu:

  • Status Flow: Antre (Kuning) -> Proses Cuci (Biru) -> Siap Ambil (Hijau) -> Selesai (Hitam).
  • Indikator Tepat Waktu:
    • üî¥ Merah: Order Telat (Overdue) atau Jatuh Tempo Hari Ini! Segera kerjakan.
    • üîµ Biru/Hijau: Order masih aman (Jatuh tempo besok atau lusa).

BAB 6: DAILY REMINDER (LAPORAN HARIAN) üö®

Fitur asisten cerdas yang muncul setiap pagi:

  • Pop-up Otomatis: Melaporkan order apa saja yang Telat dan Harus Selesai Hari Ini.
  • Lapor Owner (WA): Klik satu tombol untuk mengirim laporan ringkas ke WhatsApp Owner/Grup Tim.

  • Cetak Fisik: Mendukung Printer Thermal Bluetooth (58mm).

BAB 8: MANAJEMEN PENGELUARAN (EXPENSE) üìâ

Catat setiap uang keluar dari laci kasir (Beli sabun, Token Listrik, Makan Siang).

Pencatatan ini penting agar Profit Bersih di Dashboard akurat dan tidak semu.

BAB 9: PEMBATALAN ORDER / VOID üõ°Ô∏è

Jika terjadi kesalahan input atau pembatalan:

  • Gunakan tombol "Batalkan / Void Order" berwarna merah.
  • Wajib Isi Alasan: (Misal: "Customer Batal", "Salah Input").
  • Efek: Uang dikembalikan (minus omzet) dan Stok barang dikembalikan otomatis.

4. Aktivasi Google Sheet Sync ‚òÅÔ∏è

Fitur ini memungkinkan data order tersimpan otomatis ke Google Sheet pribadi Anda.

  1. Buka Laptop, cari file bernama KODE_GOOGLE_SHEET.txt di dalam folder aplikasi.
  2. Buka file tersebut, Copy semua kodenya.
  3. Buka browser, buat Google Sheet Baru.
  4. Klik menu Ekstensi > Apps Script.
  5. Paste kode tadi ke editor Apps Script (hapus kode bawaan).
  6. Klik Terapkan (Deploy) > Web App Baru.
  7. Pilih "Who has access" menjadi "Anyone" (Siapa saja).
  8. Copy URL Web App yang muncul.
  9. Paste URL tersebut ke kolom Sync URL di menu Pengaturan ShoeTrack.
